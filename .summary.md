- **Mobile First**: touch targets ≥44px, no horizontal scroll, responsive fonts
- **User-facing errors in Russian**; backend errors never leak stack traces
- **Security**: Zod validation, no raw SQL, JWT in httpOnly cookies, CSP headers, rate limiting on sensitive endpoints
- **Deployment**: GitHub Actions push → main → GitHub Pages with CDN cache purge
