# Руководство по разработке (Contributing Guide)

Добро пожаловать в проект **Sport Lounge**! Этот документ поможет вам настроить рабочее окружение и начать разработку.

## 🛠️ Настройка окружения

Проект представляет собой full-stack приложение (монорепозиторий):
* `/client` — фронтенд на React (Vite, TypeScript, TailwindCSS)
* `/server` — бэкенд на Node.js (Express, TypeScript, Supabase/PostgreSQL)

### Требования
* Node.js версии `>= 18.0.0`
* npm версии `>= 9.0.0`

### Шаги запуска

1. **Клонирование репозитория и установка зависимостей:**
   ```bash
   npm run install:all
   ```

2. **Настройка переменных окружения:**
   * В директории `/server` скопируйте `.env.example` в `.env` и заполните параметры (ключи Supabase, порты и т.д.).
   * В директории `/client` проверьте файлы `.env` и `.env.production` для настройки URL бэкенда.

3. **Запуск в режиме разработки:**
   ```bash
   npm run dev
   ```
   Эта команда запустит клиент и сервер параллельно с помощью пакета `concurrently`.

## 🧪 Тестирование и Сборка

### Тесты бэкенда
Тесты написаны на **Jest** и **Supertest**. Запуск тестов:
```bash
cd server
npm run test
```

### Сборка проекта
Для компиляции TypeScript кода и сборки бандлов запустите:
```bash
npm run build:all
```

## 📜 Правила кодовой базы

* Использование TypeScript обязательно для всех новых модулей.
* Для валидации форм используйте связку `react-hook-form` + `zod`.
* Тяжелые компоненты (карты, 3D сцены, плееры) загружайте лениво через `React.lazy` и `Suspense`.
* Код проходит автоматический линтинг в CI/CD пайплайнах при пуше в ветку `main`.
