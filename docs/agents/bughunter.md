# 🔍 BugHunter Agent

> Автономный агент поиска и исправления багов для SPORT LOUNGE.
> Цвет: #F44336 (красный) | Позывной: BUGBOT (B-11)

## Архитектура

```
agents/
├── agents/
│   └── bughunter/                  # Агент-демон
│       ├── agent.js                # Основной процесс (Node.js daemon)
│       ├── prompt.md               # Системный промпт
│       ├── config.json             # Конфигурация
│       └── log.md                  # Лог проверок и исправлений
├── skills/
│   └── bug-hunter-skill/
│       └── SKILL.md                # Правила, эвристики, автофиксы
```

## Запуск

Агент запускается автоматически вместе с сервером (`server.ts` → `bughunterDaemon.ts` → `agent.js`).

Ручной запуск: `node agents/agents/bughunter/agent.js`

## Как работает

1. Каждые 60 секунд сканирует `client/src/` на баги
2. Классифицирует по категориям: UI/UX, функциональные, доступность, адаптивность
3. Исправляет баги с уверенностью > 95% (автоматически)
4. Эскалирует сомнительные баги в лог и Telegram
5. Проверяет сборку (`vite build`) после каждого фикса

## Категории багов

| Категория | Примеры | Автофикс |
|-----------|---------|----------|
| `A11Y-LABEL` | Кнопка-иконка без aria-label | ✅ |
| `A11Y-ALT` | Изображение без alt | ✅ |
| `UI-GLASS` | Glassmorphism не соответствует паттерну | ✅ |
| `UI-GOLD-DUAL` | Неправильное использование #FFBF00 / #B08D57 | ✅ |
| `FUNC-EMPTY-ARR` | .map() без проверки на null | ✅ |
| `UI-COLOR` | Hardcoded цвет не из палитры | ❌ (требуется ручной фикс) |
| `FUNC-PROMISE` | Promise без .catch() | ❌ |

## Админ-панель

- **URL**: `/admin/bughunter`
- **Тумблер**: Вкл/Выкл агента
- **Статистика**: циклы, найдено, исправлено, эскалировано
- **Лог**: последние действия агента
- **Feature flag**: `bughunter_agent` в Smart Features

## Telegram

- Ежедневный отчёт: каждый день в 10:00 отправляется сводка @Yamavog
- Эскалации: при confidence < 95% баг отправляется в Telegram
