# 1C & Data Integration Skills (адаптировано из cc-1c-skills)
# Источник: https://github.com/Nikolay-Shirokov/cc-1c-skills

## Описание
Скиллы для работы с данными, интеграциями, учётными системами. Адаптировано для контекста Sport Lounge (кальянная).

## Применение в Sport Lounge

### Учёт и интеграции
Проект Sport Lounge не использует 1С напрямую, но принципы работы с данными из этого набора скиллов применимы:

#### Работа с данными
- Структурированный подход к схемам данных
- Валидация через JSON DSL
- Миграции и версионирование схем
- Import/export данных

#### Интеграция с Google Sheets
```
Учёт заказов кальянов:
- Автоматическая запись через n8n webhook
- Колонки: Дата, Время, Микс, Стоимость, Комментарий
- Итоги по дням/неделям/месяцам
- Формулы для среднего чека
```

#### Интеграция с Supabase
```
PostgreSQL для аналитики:
- Таблица orders: id, mix_id, created_at, amount, status
- Таблица analytics: daily aggregates
- Row Level Security для безопасности
- Realtime subscriptions для уведомлений
```

#### Интеграция с Telegram Bot
```
Уведомления о заказах:
- Новый заказ → сообщение в TG
- Форматирование: имя, микс, время
- Кнопки: Принять / Отклонить
```

#### n8n Workflows
```
1. Webhook: заказ с сайта
2. Supabase: сохранить в БД
3. Google Sheets: записать в таблицу
4. Telegram: уведомить владельца
5. Analytics: обновить метрики
```

### Генерация отчётов
- Дневной отчёт в Google Sheets
- Еженедельная сводка в Telegram
- Ежемесячный анализ трендов
