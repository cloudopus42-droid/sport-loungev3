# Health Checker Skill

> Навык мониторинга эндпоинтов и версий.

## 1. Возможности
- Проверка HTTP-эндпоинтов (статус, таймаут)
- Мониторинг версий сборки
- Проверка статуса всех агентов
- Telegram-алерты при падениях

## 2. Эндпоинты по умолчанию
- `/api/health` — базовый health check
- `/api/smart-features/status` — API доступность

## 3. Протокол
- GET запрос к каждому эндпоинту
- Таймаут: 10 секунд
- Проверка HTTP 200
- Измерение времени ответа
